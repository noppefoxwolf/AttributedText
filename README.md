
# AttributedText

The text rendering component like SwiftUI's Text.

# Usage

```swift
import SwiftUI
import AttributedText

Text("Hello, World!!")
DAWN.Text("Hello, World!!")
```

# Supports

- [x] NSTextAttachmentViewProvider
- [x] Self resizing
- [x] lineLimit
- [x] selection mode
- [x] extra menu actions
